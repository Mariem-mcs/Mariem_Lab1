### Lab 1B: Informed Search
**File:** `Mariem_30432027_Lab_1B.ipynb`

**Algorithms Implemented:**
- Greedy Best-First Search
- A* Search
- Weighted A* Search
- Uniform-Cost Search (UCS)
- IDA* (Bonus)

**Problem:** Drone pathfinding with:
- GPS-based heuristic (Manhattan distance)
- Non-uniform terrain costs
- Speed vs optimality trade-offs

**Key Concepts:**
- Heuristic functions (h(n))
- f(n) = g(n) + h(n)
- Admissibility and consistency
- Weighted A* trade-offs
